This scanner takes the following parameters:
  • --zap_path (-p)
    You can use this optional parameter to provide the script with (-p) the path to ZAP jar file. If you do not provide this the script tries to find the file itself and fail if it cannot find them.
  • --zap_options (-o)
    Additional command line options to provide to ZAP see (-o) 'ZAP WIKI' for more information (https://code.google.com/p/skipfish/source/browse/trunk/doc/).
  • --hosts
    Specification of hosts to scan. Follows the Nessus rules for specifying hosts, or path to a file containing such specification
  • --remote (-r)
    Option comma separed list of hostname, username, and key path used to ssh to the host and run the command there

Substitutions:

  • $HOSTS is substituted with the contents of the host field
  • $WORSKPACE is substituted with the workspace name
  • $SCAN is substituted with the scan name
  • $PASSWORD is substituted with the value in the password field
  • @HOSTS is substituted with the path of a file containig the hosts field
