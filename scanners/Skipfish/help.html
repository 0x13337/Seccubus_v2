This scanner takes the following parameters:
  • --skipfish_path (-p)
    You can use this optional parameter to provide the script with (-p) the path to Skipfish. If you do not provide this the script tries to find the files itself and fail if it cannot find them.
  • --skipfish_options (-o)
    Additional command line options to provide to Skipfish see (-o) 'skipfish -h' for more information. Please quote the options so that they are passed as a single block. E.g. -o '-option1 -option2'
  • --host
    Specification of host to scan. Can only scan one host at the time.
  • --remote (-r)
    Option comma separed list of hostname, username, and key path used to ssh to the host and run the command there

Subsitutions:

  • $HOSTS is substituted with the contents of the host field
  • $WORSKPACE is substituted with the workspace name
  • $SCAN is substituted with the scan name
